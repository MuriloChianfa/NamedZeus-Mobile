## Dependencies versions

- Docker 20.10.12
- Docker-compose 2.2.3
- NodeJS 14.21.3
- Java SDK 15.0.2
- Android 8.0 API 26

---

## Running the project

1. Run the API environment
```bash
docker compose up -d
```

2. Run the android application
```bash
npx react-native run-android
```
